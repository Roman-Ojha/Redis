-> setex <key> <expire_in_sec> <value> (to get expire data on setting value)
-> ttl <key> (time to live if '-1' then it will live for ever, '-2' for key value is expire, '<number>' expire in <number> second)
-> expire <key> <in_second> (to expire the key in certain time)
-> setnx <key> <value> (want to set the value but not override the previously set value)
-> strlen <key> (to get the length of the value)
-> mset <key1> <value1> <key2> <value2> ... <keyN> <valueN> (to set the multiple key value in once)
-> psetex <key> <expire_in_millSec> <value> (set the key value which expire in milliseconds)
-> decr <key_having_value_in_int> (decrement value)
-> incr <key_having_value_in_int> (increment value)
-> incrby <key_having_value_in_int> <increment_by> (increment value by <number>)
-> decrby <key_having_value_in_int> <decrement_by> (<decrement value by <number>)
-> append <previous_key> <value> (to append in previously set value)
